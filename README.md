## 🚀 Quick Start

### Prerequisites
- **Node.js** 18.x or higher
- **npm** or **yarn** package manager

### Installation

```bash
# Clone the repository
git clone https://github.com/Dhatripatel06/pf.git
cd pf

# Install dependencies
npm install

# Start development server
npm run dev
```

The portfolio will be available at `http://localhost:5173`

---

## 📦 Build & Deploy

```bash
# Build for production
npm run build

# Preview the production build
npm run preview

# Development build (with debug info)
npm run build:dev
```

The optimized build is output to the `dist/` directory and ready for deployment to Vercel, Netlify, or any static hosting service.

## 🛠️ Tech Stack

### Frontend
- **Framework**: React 18.3 with TypeScript
- **Build Tool**: Vite 5.4 (Lightning-fast builds)
- **Styling**: Tailwind CSS + shadcn/ui components
- **Animations**: Framer Motion for smooth, interactive elements
- **Routing**: React Router v6
- **3D Graphics**: Three.js + React Three Fiber (for immersive visuals)

### State Management & Data
- **Query Management**: TanStack React Query v5
- **Form Handling**: React Hook Form with Zod validation
- **UI Components**: Radix UI (39+ accessible components)

### Additional Libraries
- **Charting**: Recharts for data visualization
- **Toasts**: Sonner for notifications
- **Icons**: Lucide React (500+ icons)
- **Carousel**: Embla Carousel for image galleries
- **OTP Input**: Input-OTP for authentication flows
- **Date Handling**: Date-fns
- **Utility**: Tailwind Merge, CLSX for class management

### Development Tools
- **Testing**: Vitest with React Testing Library
- **Linting**: ESLint with React Hook Rules
- **Post-CSS**: Autoprefixer for browser compatibility
- **TypeScript**: v5.8+ for type safety
